# WWW Backend

Backend for www project

## Installation

- First copy the `example.env` file to `.env` file

    ```bash
    cp example.env .env
    ```

- Use `yarn` to install the dependencies.

    ```bash
    yarn
    ```

- Generate the prisma types with `prisma`

    ```bash
    npx prisma generate
    ```

## Quick Start

- Use the default scripts to run the project

    ```bash
    yarn dev
    ```
